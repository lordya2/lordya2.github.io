## Google Analytics 4

This site supports optional Google Analytics 4 tracking through `_config.yml`.
To enable analytics, set `google_analytics_id` to a valid GA4 measurement ID, for example `G-XXXXXXXXXX`.
Leave the value empty to prevent the analytics script from loading.

Important homepage links include `data-analytics-event` attributes for CV downloads, publication clicks, student-opportunity clicks, email clicks, and external-profile clicks so GA4 events can be configured without changing content links.
